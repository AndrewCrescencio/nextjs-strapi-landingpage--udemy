## NextJs-Strapi-Landing Page

Está é uma aplicação de uma landing page gerada com Server Side Rendering do NextJs com TypeScript, e estilizada usando Styled Components. O backend é feito no CMS Strapi, então todo o conteúdo pode ser facilmente manipulado por lá.
